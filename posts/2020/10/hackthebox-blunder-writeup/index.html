HackTheBox- Blunder Writeup

Machine IP : 10.10.10.191

This following write up is for the machine: Blunder

Hack the Box is an online CTF platform where you can hone your penetration testing skills. It’s similar to boot2root machines. The goal is to find two flags namely, User flag and Root flag to successfully root the machine.

Synopsis

Blunder is the easy boot2root machine where we need to find two flags in the machine. Enumeration is the key and this machines does require a bit of enumeration and exploiting the specific CMS version and then getting the initial foothold, later it becomes easy and escaltion to root is very simple could be done by popular SUDO exploit.

Post- exploitation

So, lets enumerate the machine. So, i have tried sudo -l and immediately, we got the answer and solution to get to the root.

Once, I saw this, I know at once its CVE-2019-14287, SUDO flaw which allows the users to run command even in the restricted access.
